WebRTC
실시간 통신 솔루션

브라우저만 있으면 연결됩니다. 설치 없이, 실시간으로. WebRTC는 플러그인 설치 없이 웹 브라우저 상에서 음성, 영상, 데이터 공유가 가능한 차세대 웹 표준 기반 실시간 통신 기술입니다.
CEN은 WebRTC를 기반으로, 무인 관제, 원격 지원, AI 기반 영상 분석 연동, IoT 디바이스 제어까지 확장 가능한 솔루션을 제공합니다.
