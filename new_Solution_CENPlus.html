스마트시티 통합플랫폼 개발

도시는 연결되고, 플랫폼은 진화합니다.



CENPlus는 씨이엔코퍼레이션이 자체 개발한 스마트시티 통합 운영 플랫폼으로,
도시 전역에 설치된 IoT 센서와 시스템, AI 분석, 공공 데이터까지 하나로 통합하여 효율적인 도시 운영과
시민 안전, 에너지 최적화, 디지털 행정 서비스를 가능하게 합니다.

"스마트 건설의 중심, 자율과 안전을 설계하다"

주요 기술

Apache Kafka Cluster 기반 구축

데이터 스트림을 실시간 처리하여 플랫폼 내외적으로 다양한 곳에서 발생되는 데이터 정보들을 한 데 모아
일률적으로 관리하여 처리하는 데이터 허브 역할의 플랫폼을 구축하고 구현합니다.

고성능 확장성

데이터를 순차적으로 처리하는
고성능 파이프라인은 분산 시스템의
확장성을 높입니다.

내결합성

디스크 저장과 클러스터 복제로
데이터 내구성과 접근성을
확보합니다.

연계 통합의 데이터 허브

데이터 연계와 복제를 지원하며
통합 스토리지 관리를
제공합니다.

MQTT IoT Broker 기반 실시간 서비스 제공

Kafka 클러스터와 데이터베이스 간 파이프라인을 제공하는 Kafka Connect 및 Kafka Connect 와
Pub-Sub 모델의 경량 메시징 프로토콜인 MQTT를 사용하여 이벤트, 소통정보 등의 데이터를 실기간으로
전달 처리하고, 관련 서비스를 제공합니다.
